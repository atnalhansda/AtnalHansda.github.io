Overview

OL Chiki IME lets you type Ol Chiki script (Ol Chiki/Bengali script mapping available) across Windows apps — Word, Notepad, browsers and more. Designed for low-latency typing and compatibility with modern Windows versions (10/11).

Unicode-compliant

Full Unicode support for Ol Chiki characters (U+1C50–U+1C7F).

Installer

Simple EXE installer for quick setup.

Code-signed

Signed installer to reduce SmartScreen warnings and improve trust.

Installation

  1. Download the installer below.
  2. Run the installer as Administrator and follow prompts.
  3. Open Settings → Time & Language → Language → Add a keyboard → select "OL Chiki IME" if not auto-added.
  4. Open an application and switch input method to OL Chiki to start typing.
