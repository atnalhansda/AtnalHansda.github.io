Ol Chiki IME Software

Ol Chiki IME is a lightweight input method editor that allows users to type in the Ol Chiki script (ᱚᱞ ᱪᱤᱠᱤ), the writing system of the Santali language. It works seamlessly across applications like MS Word, browsers, and social media.

  • ✔ Full Ol Chiki script support with Unicode compliance
  • ✔ Phonetic keyboard for easy typing
  • ✔ Works in MS Word, Excel, browsers, and more
  • ✔ Compatible with Ol Chiki fonts (e.g., Nirmala UI)
  • ✔ Lightweight, fast, and easy to install
